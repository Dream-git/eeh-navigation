What is this all about?

The eeh-menu AngularJS module provides a configurable, responsive navbar and sidebar directive. It also contains a companion menu item configuration service. The directives support navigation of UI-Router states, ng-click binding, and ng-href links.

eeh-menu is used in its own documentation (i.e. this site).

There are live examples of eeh-menu in action.

The AngularJS version of the SB Admin 2 theme also uses eeh-menu.
