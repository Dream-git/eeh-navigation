eeh-navigation-navbar

  1. - directive in module

This directive allows for the creation of a Twitter Bootstrap navbar component.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <eeh-navigation-navbar
  [menu-name="string"]
  [nav-class="string"]
  [container-class="string"]
  [brand-text="string"]
  [brand-state="string"]
  [brand-href="string"]
  [brand-target="string"]
  [brand-src="string"]
  [brand-click=""]
  [refresh=""]>
...
</eeh-navigation-navbar>
  • as attribute: 
    <ANY
  [menu-name="string"]
  [nav-class="string"]
  [container-class="string"]
  [brand-text="string"]
  [brand-state="string"]
  [brand-href="string"]
  [brand-target="string"]
  [brand-src="string"]
  [brand-click=""]
  [refresh=""]>
...
</ANY>

Arguments

Param Type Details
menuName
(optional)
 string

Sets the name of the menu that the directive will render.
navClass
(optional)
 string

Sets the ng-class attribute of the top-level nav element.

(default: navbar-default navbar-static-top)
containerClass
(optional)
 string

Sets the ng-class attribute of the top-level "container" element. Common settings are nothing, "container" or "container-fluid". Some custom class or expression can also be used if desired.
brandText
(optional)
 string

Sets the text of the brand element.
brandState
(optional)
 string

Sets ui-sref of the brand element.
brandHref
(optional)
 string

Sets the href attribute of the brand element.
brandTarget
(optional)
 string

Sets target attribute of the brand element.
brandSrc
(optional)
 string

Sets the src attribute of the image in the brand element.
brandClick
(optional)
 function=

Sets the callback function of the brand element.
refresh
(optional)
 function=

A function for refreshing the directive.
